### Redis Streams란?

Redis Streams는 **추가 전용(append-only) 로그 데이터 구조**입니다. Kafka와 유사하게 메시지 큐, 이벤트 소싱, 실시간 데이터 처리에 활용됩니다.

각 엔트리는 다음으로 구성됩니다:
- **고유 ID**: `<밀리초타임스탬프>-<시퀀스번호>` 형식
- **필드-값 쌍**: 하나 이상의 키-값 데이터

## 📚 난이도 설명

- ⬤○○ **쉬움**: 기본 개념 이해와 간단한 구현
- ⬤⬤○ **보통**: 여러 개념의 조합 또는 약간 복잡한 로직
- ⬤⬤⬤ **어려움**: 동시성 처리, 비동기 로직 등 고급 기능
